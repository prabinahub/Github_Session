How to Use GitHub

Guide to Start with GitHub

Step-by-Step Process
  1. Create a GitHub Account: Go to github.com and sign up for a free account.
  2. Install Git: Download Git from git-scm.com and install it on your computer.
  3. Set Up Git: Configure your name and email using these commands: 
    git config --global user.name "Your Name"
git config --global user.email "your@email.com"
  4. Create a Repository: On GitHub, click the "+" icon â†’ "New repository". Give it a name and description.
  5. Clone the Repository: Copy the repo URL and use: 
    git clone https://github.com/your-username/your-repo-name.git
  6. Add Files: Create or add files in the cloned folder.
  7. Stage Changes: Use: 
    git add .
  8. Commit Changes: Use: 
    git commit -m "Initial commit"
  9. Push to GitHub: Use: 
    git push origin main
  10. Check GitHub: Visit your GitHub repo to see the uploaded files.
